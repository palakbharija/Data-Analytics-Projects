# Task 1 - Data Immersion and Wrangling

## Overview

This project focuses on understanding, cleaning, and preparing a sales dataset for further analysis.

The dataset was examined to identify data quality issues such as missing values, duplicate records, inconsistent data, and formatting issues. The data was then cleaned and transformed using Python and Pandas.

## Dataset

The dataset contains sales transaction information, including:

- Order ID
- Order Date
- Customer ID
- Customer Name
- Age
- Gender
- City
- Product
- Category
- Quantity
- Unit Price
- Total Sales

## Data Cleaning and Transformation

The following steps were performed:

- Explored and understood the dataset structure
- Checked for missing values
- Checked for duplicate records
- Handled missing values
- Checked and corrected data types
- Performed data transformations
- Created new derived columns such as Month and Age Group
- Prepared the cleaned dataset for further analysis

## Project Files

- `ApexPlanet_DataAnalytics_Dataset.xlsx` – Original dataset
- `Cleaned_Sales_Dataset.xlsx` – Cleaned and transformed dataset
- `Data_Cleaning.ipynb` – Python notebook containing the data cleaning process
- `data_dictionary.xlsx` – Description of dataset columns

## Tools Used

- Python
- Pandas
- Jupyter Notebook
- Microsoft Excel

## Outcome

The dataset was successfully cleaned, transformed, and prepared for further exploratory data analysis and business intelligence tasks.
